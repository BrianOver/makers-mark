# 3D Town Hub + Interaction — Implementation Plan (rev. 2, post Fable review)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the 2D floating-sprite town hub with a grounded true-3D low-poly town (generic Kenney CC0 meshes): the player avatar walks (WASD + click-to-move), buildings read as real grounded structures, interaction is obvious (proximity highlight + prompt → existing panels), and the menu-sizing defects are fixed.

**Architecture:** A `Node3D` world lives inside a `SubViewport` (hosted by a `SubViewportContainer`, `Stretch=true`) that becomes the base child of `main_ui.tscn`'s Control tree; the 2D HUD/`DrawerHost` draw on top. The world reads `SimAdapter.CurrentState` and re-emits the existing `BuildingClicked(string)`/`HeroClicked(int)` vocabulary into `MainUi` — zero sim change (KTD2). Real 3D depth replaces Y-sort. `NavigationAgent3D` (baked from a finite ground box + building footprint colliders) drives click-to-move; nav + positions are presentation-only.

**Build strategy (why the task order):** the 3D town is built as **standalone** scripts + tests (T3–T7) that mount `Town3D` directly in a code-built rig — **nothing touches `MainUi` and nothing is deleted**, so the build and the full engine suite stay green after every task. A single **atomic cutover** (T8) then wires `Town3D` into `MainUi`, relocates shared statics, and deletes the 2D town + its tests together. This is the sequence the review requires — do not swap `MainUi` early.

**Tech Stack:** Godot 4.6.3-stable .NET (C#, `net10.0` in `godot/GodotClient.csproj`), gdUnit4Net (`gdUnit4.api 5.0.0`), Kenney CC0 GLB via Git LFS.

```csharp
using Godot;
namespace GodotClient.Town3d;
public partial class CameraRig : Node3D
{
    [Export] public Node3D? Target;
    [Export] public float Pitch = -50f;
    [Export] public float Distance = 14f;
    [Export] public float FollowSpeed = 5f;
    private Camera3D _cam = null!;
    public override void _Ready()
    {
        _cam = new Camera3D { Name = "Camera3D", Fov = 45f, Near = 0.5f, Far = 200f };
        AddChild(_cam);
        _cam.RotationDegrees = new Vector3(Pitch, 0, 0);
        _cam.Position = _cam.Basis.Z * Distance;
        _cam.Current = true;
        if (Target != null) GlobalPosition = Target.GlobalPosition;
    }
    public override void _Process(double delta)
    {
        if (Target == null) return;
        float t = 1f - Mathf.Exp(-FollowSpeed * (float)delta);
        GlobalPosition = GlobalPosition.Lerp(Target.GlobalPosition, t);
    }
}
```
